This crate creates checkpoints on disk for an optimization run.

Saves a checkpoint on disk from which an interrupted optimization run can be resumed. For details on the usage please see the documentation of FileCheckpoint or have a look at the example.

§Usage

Add the following line to your dependencies list:

[dependencies]
argmin-checkpointing-file = "0.1.0"
