Levy test function

Defined as

f(x_1, x_2, ..., x_n) = sin^2(pi * w1) + \sum_{i=1}^{d-1}(w_i -1)^2 * (1+10*sin^2(pi*wi+1)) + (w_d - 1)^2 * (1 + sin^2(2*pi*w_d))

where w_i = 1 + (x_i - 1)/4 and x_i \in [-10, 10].

The global minimum is at f(x_1, x_2, ..., x_n) = f(1, 1, ..., 1) = 0.
